# Small User Study Plan

I want to test whether this assistant actually helps students work with lecture PDFs, instead of judging it only from my own test questions.

## Research question

Can a source-grounded document assistant help students find and verify information in lecture material more quickly while keeping unsupported answers visible?

## Planned participants

- 5 to 10 university students
- Participation is voluntary
- Only non-sensitive lecture material should be used
- No names are needed in the results

## Tasks

Each participant will:

1. Upload one lecture PDF.
2. Find the explanation of one concept.
3. Ask one factual question whose answer is present.
4. Ask one question whose answer is absent.
5. Verify the answer against the displayed source or PDF page.
6. Rate whether the answer was helpful and add a correction if needed.

## Measurements

- Task completion time
- Whether the correct source was retrieved
- Whether unsupported questions were refused
- Helpful / not-helpful ratings
- Number of corrections
- Self-reported confidence before and after source checking

## Short survey

Participants can rate each statement from 1 (strongly disagree) to 5 (strongly agree):

1. I could find the relevant information quickly.
2. The displayed source made the answer easier to verify.
3. The answer was clear enough for study or revision.
4. I understood when the assistant was uncertain.
5. I would use this tool with my own lecture material.

An optional free-text question asks: "What was confusing or missing?"

## Reporting

The README will only include results after the study is completed. Planned sample sizes, expected improvements, or benchmark scores will not be presented as user-study findings.
